# ГОСТ Р 34.10-2018 — Электронная цифровая подпись

Реализация процессов **формирования** и **проверки** электронной цифровой подписи (ЭЦП) на языке Python в соответствии с ГОСТ Р 34.10-2018, с использованием хэш-функции «Стрибог» (ГОСТ Р 34.11-2018).

---

## Стандарты

| Стандарт | Назначение |
|---|---|
| ГОСТ Р 34.10-2018 | Алгоритм ЭЦП на эллиптических кривых |
| ГОСТ Р 34.11-2018 | Хэш-функция «Стрибог» (Streebog) |
| Р 1323565.1.024-2019 | Параметры эллиптических кривых ТС26 |

## Быстрый старт

```python
from gost_sign import GOST34102018

# Создать объект с режимом 256 бит
gost = GOST34102018(mode=256)

# Сгенерировать ключи
keys = gost.generate_keys()

# Подписать сообщение
message = "Мой документ".encode("utf-8")
sig = gost.sign(message, keys.private_key)

# Проверить подпись
ok = gost.verify(message, sig.signature, keys.public_key)
print("Подпись верна:" if ok else "Подпись неверна!")
```

## API

### Класс `GOST34102018`

```python
GOST34102018(mode=256, curve_id=None)
```

| Параметр | Тип | Описание |
|---|---|---|
| `mode` | `int` | Режим: `256` или `512` бит |
| `curve_id` | `str` | ID кривой (если `None` — выбирается по умолчанию) |

#### Методы

| Метод | Описание | Возвращает |
|---|---|---|
| `generate_keys()` | Генерация пары ключей | `KeyPair` |
| `hash_message(message)` | Хэширование Стрибогом | `bytearray` |
| `sign(message, private_key)` | Формирование подписи | `SignatureResult` |
| `verify(message, signature, public_key)` | Проверка подписи | `bool` |
| `info()` | Текстовое описание параметров | `str` |

---

### Класс `KeyPair`

Хранит пару ключей. Поля:

| Поле | Тип | Описание |
|---|---|---|
| `private_key` | `bytearray` | Приватный ключ |
| `public_key` | `bytearray` | Открытый ключ |
| `mode` | `int` | Режим (256 / 512) |
| `curve_id` | `str` | ID кривой |
| `private_hex` | `str` | Приватный ключ в hex |
| `public_hex` | `str` | Открытый ключ в hex |

---

### Класс `SignatureResult`

Результат подписания. Поля:

| Поле | Тип | Описание |
|---|---|---|
| `signature` | `bytearray` | Подпись (r, s) |
| `digest` | `bytearray` | Хэш сообщения |
| `mode` | `int` | Режим (256 / 512) |
| `hash_alg` | `str` | Алгоритм хэширования |
| `timestamp` | `str` | Метка времени (UTC, ISO 8601) |
| `signature_hex` | `str` | Подпись в hex |
| `digest_hex` | `str` | Хэш в hex |

## Режимы работы

| Режим | Кривая по умолчанию | Хэш | Размер ключа | Размер подписи |
|---|---|---|---|---|
| 256 бит | ТС26 256-A | Стрибог-256 | 32 байт | 64 байт |
| 512 бит | ТС26 512-B | Стрибог-512 | 64 байт | 128 байт |

### Доступные кривые

**256 бит:**
- `id-tc26-gost-3410-2012-256-paramSetA` *(по умолчанию, КС1/КС2)*
- `id-tc26-gost-3410-2012-256-paramSetB`
- `id-tc26-gost-3410-2012-256-paramSetC`
- `id-tc26-gost-3410-2012-256-paramSetD`

**512 бит:**
- `id-tc26-gost-3410-12-512-paramSetB` *(по умолчанию, КВ1/КВ2)*
- `id-tc26-gost-3410-12-512-paramSetA`
- `id-tc26-gost-3410-2012-512-paramSetC`

## Сохранение и загрузка

```python
from gost_sign import save_keys, load_keys, save_signature, load_signature

# Сохранить ключи
save_keys(keys, "gost_keys.json")

# Загрузить ключи
keys = load_keys("gost_keys.json")

# Сохранить подпись
save_signature(sig, "gost_sig.json")

# Загрузить подпись и проверить
sig = load_signature("gost_sig.json")
ok = gost.verify(message, sig.signature, keys.public_key)
```

Пример сохранённых ключей (`gost_keys.json`):

```json
{
  "mode": 256,
  "curve_id": "id-tc26-gost-3410-2012-256-paramSetA",
  "private_key": "be79638814d947c5...",
  "public_key": "4874259229b31ee8..."
}
```

> ⚠️ **Приватный ключ** включается в файл — храните его в защищённом месте и никогда не передавайте третьим лицам.

## Алгоритм (кратко)

### Формирование подписи

```
M  →  e = Стрибог(M)
       ↓
k  — случайное число из [1, q-1]
       ↓
C  = k · P   →   r = x_C mod q
       ↓
s  = (r·d + k·e) mod q
       ↓
Подпись = (r, s)
```

### Проверка подписи

```
M', (r, s), Q
       ↓
e' = Стрибог(M')
       ↓
v  = e'^(-1) mod q
z1 = s·v mod q,   z2 = -r·v mod q
       ↓
A  = z1·P + z2·Q   →   R = x_A mod q
       ↓
Верна ⟺ R == r
```

## Запуск демонстрации

```bash
python gost_sign.py
```

Демонстрация проверяет:
- подпись корректного сообщения (должна быть верна)
- подпись изменённого сообщения (должна быть отклонена)
- подпись с чужим ключом (должна быть отклонена)
- режим 512 бит
- сохранение и загрузку ключей / подписи из файла
